# Inventory Management System (Java)

## Overview
This is a comprehensive desktop application developed as part of my **Graded Unit (Level 8)** at the University of the West of Scotland. The system is designed to manage retail operations, inventory tracking, and staff management with a focus on security and data integrity.

## Features
- **Role-Based Access Control:** Distinct interfaces and permissions for Managers, Sales Associates, and Warehouse Staff.
- **Inventory Logic:** Real-time stock management, order fulfillment, and automated calculations.
- **Modern UI:** Built with **JavaFX** and FXML for a responsive and intuitive user experience.
- **Robust Error Handling:** Custom exception handling to ensure system stability during invalid data input.

## Tech Stack
- **Framework:** JavaFX
- **Tools:** IntelliJ IDEA, Scene Builder
- **Concepts:** Object-Oriented Programming (Inheritance, Polymorphism, Encapsulation), File I/O.
